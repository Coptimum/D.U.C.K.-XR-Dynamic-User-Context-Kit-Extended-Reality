ENDGOAL: NeuroSwarm 
Focus Topics: A device that combines and analyzes various biomarkers, to observe potential changes in social interaction
Consequences: Neuroethics
So now we are pivoting towards inverting our project so that the user of the headset is able to access real time biomarkers as well as their own. A program in the background will be running an analysis on all these biomarkers and relaying them back to the user via Mixed Reality (MR). (We will decide through testing and feasibility what type of information will actually be relayed back to the user I guess) as well as trends that could be found through these biomarkers. We want to analyze not only the biomarkers, but how a person who has access to all this Biodata changes their behaviors or interaction in a social setting. 
Phase 1 - implementation of all these features
Phase 2 - Making the ecosystem of the entire project (e.g. the actual integration of MR and the biodata that we receive from these features) 
Phase 3 - analysis on impacts of these features (social experiment-esque study)
